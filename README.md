## 注意点

- 本リポジトリ及びページについては、公開されているため機密情報の取り扱いにはご注意ください。
  - 個人情報、パスワードやクレジットカード番号といった機密情報のアップ、コメントは行わない。
  - 社外秘資料等のアップ、コメントは行わない。
- 公開されたGitHub Pagesのサイトは1GB以上であってはなりません。
- GitHubは100 MB以上のプッシュをブロックします。
