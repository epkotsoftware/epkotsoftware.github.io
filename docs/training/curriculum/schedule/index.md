# 新人研修　〜　スケジュール

## スケジュール

本資料は、IT未経験者の場合のスケジュールになります。  

## 案件の判断について

研修を通じて、講師の判断で案件（プロジェクト）の方向性を決めます  
1対1PG研修後に最終決定します。

| プログラマー適正 | 案件 | 備考 |
| :---: | :---: | :---: |
| 低 | テスト |  |
| 中 | テスト or 開発 | 30歳近い場合、開発案件から |
| 高 | 開発 | 20代前半の場合、テストからでも可 |

- 判定材料
  - 提出物の質（勤務表、課題等）
  - 研修進捗
  - 読解力
  - 説明力
  - 論理的思考力
  - コミュニケーション力
  - メンタル
  - 業務態度
