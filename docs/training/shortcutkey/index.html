VSCodeショートカット

Windows
ショートカットキー		 macOS
ショートカットキー		 操作 コマンドID
F1 F1 すべてのコマンドの表示 workbench.action.showCommands
F2 F2 シンボルの名前変更 editor.action.rename
Ctrl+L ⌘(command)+L 行全体を選択する expandLineSelection
単語をダブルクリック
またはCtrl+D		 単語をダブルクリック
または⌘(command)+D		 単語が選択状態になる -
Ctrl+D ⌘(command)+D 選択した項目を次の一致項目に追加 editor.action.addSelectionToNextFindMatch
Ctrl+Click
(対象をクリック)
またはF12
(対象にカーソルを置いた状態) 		⌘(command)+Click
(対象をクリック)
またはF12
(対象にカーソルを置いた状態) 		定義へ移動 editor.action.revealDefinition
Alt+←(LeftArrow) ⌃(control)+- 前に戻る workbench.action.navigateBack
Alt+→(RightArrow) ⌃(control)+_ 次に進む workbench.action.navigateForward
Alt+↑(UpArrow) ⌥(option)+↑ 行を上へ移動 editor.action.moveLinesUpAction
Alt+↓(DownArrow) ⌥(option)+↓ 行を下へ移動 editor.action.moveLinesDownAction
Shift+Alt+↑(UpArrow) ⇧(shift)+⌥(option)+↑ 行を上へコピー editor.action.copyLinesUpAction
Shift+Alt+↓(DownArrow) ⇧(shift)+⌥(option)+↓ 行を下へコピー editor.action.copyLinesDownAction
Ctrl+Alt+↑(UpArrow) ⌥(option)+⌘(command)+↑ カーソルを上に挿入 editor.action.insertCursorAbove
Ctrl+Alt+↓(DownArrow) ⌥(option)+⌘(command)+↓ カーソルを下に挿入 editor.action.insertCursorBelow
Ctrl+W ⌘(command)+W 表示: エディターを閉じる workbench.action.closeActiveEditor
Ctrl+P ⌘(command)+P ファイル名に移動… workbench.action.quickOpen
Ctrl+Shift+F ⇧(shift)+⌘(command)+F 表示: 検索 を表示 workbench.view.search
Ctrl+F ⌘(command)+F 検索 actions.find
  • 「定義へ移動」、「前に戻る」、「次に進む」はセットで覚えておくと便利です。
